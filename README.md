# Tesla Turbine.Hybrid Electric.Vehicle
Tesla turbine with a combined cycle **Supercritical CO2 Cycle** and a bottoming [Brayton cycle](https://en.wikipedia.org/wiki/Brayton_cycle) is the perfect powerplant for a Hybrid Electric Vehicle! Have a very small battery &amp; Supercapacitor bank to absorb loads / regenerative braking. And a single transmission-less Flux Switching Alternator.
